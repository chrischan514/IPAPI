# IP-API Tools
Quick Tools for enquiring IP Address' Data

## Usage
```bash
python3 ipapi.py -h
```
```
usage: ipapi.py [-h] [-i] [-v]

IPAPI

optional arguments:
  -h, --help  show this help message and exit
  -i          Enable DNS proxy
  -v          Verbose mode
```

To quit the script, type
```exit``` or ```exit()```
